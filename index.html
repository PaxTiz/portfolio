Site web
Refonte du site web iut-informatique.fr
HTML & CSS

Site web
Site web de référence de séries
HTML/CSS & PHP (Laravel)
Pour le projet du semestre 3 à l'IUT de Lens, nous avions à réaliser un site internet qui permet de référencer des séries. Il a été réalisé en PHP avec le framework Laravel et inclue diverses fonctionnalitées telles que la création de compte, l'ajout de séries à une liste privée, indiquer que nous avons vu un épisode, une saison ou une série en entière. On peut également consulter les différentes statistiques.

Jeu
Jeu du style "Bomberman"
Java & JavaFX
En tant que projet du semestre 2 de DUT Informatique, nous avions dû réaliser un jeu en groupe de 2. Celui-ci s'inspire du Bomberman originel. Les technologies utiliséee sont Java et JavaFX. Il y a un menu principal, un menu de pause, un menu de mort ainsi que le jeu en lui-même, et il est même possible de recommencer une partie.
